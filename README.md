# AI Multimodal Image Assistant

A free multimodal AI project that generates descriptions for images using Python and Streamlit.

## Features
- Upload any image and get an AI-generated description
- Built with Transformers (BLIP model) for image captioning
- Easy-to-use web interface via Streamlit

## Technologies
- Python 3.x
- Streamlit
- Transformers
- PIL (Python Imaging Library)

## How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/rahulrr714-lang/ai-multimodal-image-assistant.git

Go into the project folder:

cd ai-multimodal-image-assistant


Install dependencies:

pip install -r requirements.txt


Run the app:

streamlit run app.py


Open the URL shown in your browser (usually http://localhost:8501)
